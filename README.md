# godot_play_tool

<br>

## 개괄 | Overview
- 키보드 입력으로 Scene 전환, Action 수행을 도와주는 도구
- 점, 원, 사각형 등을 그려주는 도구
- code 출력, log 출력, volume 조절 도구

<br>

## 작업환경 | Environment
- OS
  - Windows 10

- Engine
  - Godot v4.4 Stable Official
